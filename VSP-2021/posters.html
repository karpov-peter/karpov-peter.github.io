Some instructions for the poster:

We kindly ask you to send your poster as a single page in png format (pdf-format is not recommended but acceptable, see below). Conversion to png can be done as follows:

PowerPoint: File -> Export -> Change file type -> PNG.

Keynote: File -> Export to -> Images -> Format -> PNG.

LibreOffice: File -> Export -> Filter:PNG

If you have troubles with this, please forward us your poster in your format and we’ll try to convert it for you.

Please use the following convention for the file name: "FirstName FamilyName, Title.png".

Example: "Peter Karpov, Quantum droplet phases in extended Bose-Hubbard model.png"
